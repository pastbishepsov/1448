# 14:48 — Admin Panel (React)

Веб-интерфейс для администраторов и владельцев клубов.

## Стек
- **React 19** + TypeScript + Vite
- **shadcn/ui** (компоненты)
- **TanStack Query** (data fetching)
- **Recharts** (графики для Owner Stats)
- **WebSocket** (real-time карта ПК)

## Части
- **Admin Panel** — управление гостями, ПК, бронями
- **Owner Stats** — аналитика, выручка, поведение игроков

## Старт
Взять как основу: `github.com/d3george/slash-admin`
Или react-admin фреймворк: `marmelab.com/react-admin`
